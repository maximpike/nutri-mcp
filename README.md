# NutriMCP - Intelligent Meal Planning Assistant

A Java Spring Boot application that integrates with Anthrotropic's Model Context Protocol (MCP) to provide intelligent meal planning based on nutritional data.

### Architecture Overview
- **Backend**: JAva Spring Boot
- **MCP Integration**: Custom MCP server for food/nutrition APIs
- **Infrastructure**: AWS (Terraform + Ansible)
- **Database**: PostgreSQL
